Overview

A program written in C++ to generate a report card. It was programmed by Alexis Ayuso.

Goal

Allows the user to enter student information and course information to calculate their GPA. The grade point table by default is from National Dong Hwa University.

  2. Add Course

  4. Delete Course
  5. View Grading System
  6. Export Report Card
  7. Quit
